# LangChain-JS

This repository are a series of demonstration scripts highlighting the functionalities of LangChain, a JavaScript library tailored for developing conversational AI applications.

- `0-basics.js`: Presents the fundamentals of utilizing the OpenAI API independently of LangChain.
- `1-first_chain.js`: Exhibits the process of establishing the initial conversation chain using LangChain.
- `2-simplesequentialchain.js`: Offers a straightforward example of constructing a sequential conversation chain.
- `3-sequentialchain.js`: Provides a comprehensive guide to creating and employing a sequential conversation chain in LangChain.
- `4-parsers.js`: Demonstrates the implementation of parsers for processing input and output within a conversation chain.
- `5-indexes.js`: Expounds on the creation and utilization of indexes in LangChain for efficient data retrieval.
- `6-usestore.js`: Offers guidance on leveraging Vector Databases in Langchain for storing and retrieving additional information not present in the model.
- `7-chat.js`: Showcases the creation of a chat bot in LangChain, serving as a foundational element for conversational AI applications.
- `8-agents.js`: Illustrates the creation and utilization of agents in LangChain, autonomous entities capable of interacting within a conversation chain.

To execute these examples, clone the git repository and execute `pnpm install` to install the required dependencies. Additionally, create a .env file and insert your OpenAI API Key in the format: `OPENAI_API_KEY=sk-....`

These codes leverage ES6 modules, enabling `import` statements and utilizing `async/await` functionalities within NodeJS.
